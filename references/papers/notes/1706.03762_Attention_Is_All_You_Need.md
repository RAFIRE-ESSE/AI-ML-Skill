## Key Innovation

Replaces recurrent (RNN/LSTM) and convolutional (CNN) sequence architectures entirely with a novel architecture based purely on **Self-Attention** and **Multi-Head Attention** mechanisms. This enables massive parallelization during training and models long-range dependencies in constant $O(1)$ operations per sequence pair.

---

## Core Formulations

### Scaled Dot-Product Attention
$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$
- Scaling factor $\sqrt{d_k}$ prevents inner products from growing large for large dimensions, which would push Softmax into regions with extremely small gradients.

### Multi-Head Attention
$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, \dots, \text{head}_h)W^O$$
$$\text{where } \text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$$

### Positional Encoding
Since Transformers contain no recurrence or convolution, spatial order is injected via sinusoidal positional encodings:
$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$
$$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

---

## Architecture Overview

1. **Encoder Stack**: 6 identical layers, each with 2 sub-layers:
   - Multi-Head Self-Attention.
   - Position-wise Feed-Forward Network ($\text{FFN}(x) = \max(0, xW_1 + b_1)W_2 + b_2$).
   - Residual connection around each sub-layer followed by Layer Normalization (`LayerNorm(x + SubLayer(x))`).
2. **Decoder Stack**: 6 identical layers with 3 sub-layers:
   - Masked Multi-Head Self-Attention (prevents attending to future positions).
   - Cross Multi-Head Attention over Encoder output.
   - Position-wise Feed-Forward Network.

## Code Reference (PyTorch)

```python
import torch
import torch.nn as nn
import math

class ScaledDotProductAttention(nn.Module):
    def __init__(self, d_k):
        super().__init__()
        self.scale = 1.0 / math.sqrt(d_k)

    def forward(self, q, k, v, mask=None):
        scores = torch.matmul(q, k.transpose(-2, -1)) * self.scale
        if mask is not None:
            scores = scores.masked_fill(mask == 0, -1e9)
        attn = torch.softmax(scores, dim=-1)
        return torch.matmul(attn, v)
```
