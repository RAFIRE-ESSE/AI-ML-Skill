## 1. Key Innovation & Algorithmic Breakthrough

Replaces positional embeddings with a static linear bias penalty directly added to attention score matrices, enabling zero-shot context length extrapolation.

---

## 2. Core Mathematical Formulations

$$A_{ij} = q_i k_j^T - m \cdot |i - j|$$

---

## 3. Architecture & Operational Mechanics

Head-specific fixed slope penalty $m = 2^{-\frac{8i}{h}}$ subtracting distance penalties from query-key dot products.

---

## 4. Code Implementation Reference

```python
def get_alibi_biases(num_heads, max_seq_len):
    slopes = torch.tensor([2**(-8 * i / num_heads) for i in range(1, num_heads + 1)])
    distance_matrix = torch.arange(max_seq_len).unsqueeze(0) - torch.arange(max_seq_len).unsqueeze(1)
    return -slopes.unsqueeze(1).unsqueeze(2) * distance_matrix.abs()
```
