## 1. Key Innovation & Algorithmic Breakthrough

Evaluates Gated Linear Unit (GLU) variants in Transformer Feed-Forward Networks, establishing SwiGLU as superior to standard ReLU/GELU FFNs.

---

## 2. Core Mathematical Formulations

$$\text{SwiGLU}(x) = (x W + b) \otimes \text{SiLU}(x V + c) W_2$$

---

## 3. Architecture & Operational Mechanics

Replaces standard 2-layer FFN $(W_2 \cdot \text{ReLU}(W_1 x))$ with element-wise gated bilinear projections.

---

## 4. Code Implementation Reference

```python
class SwiGLUFFN(torch.nn.Module):
    def __init__(self, d_model, d_ff):
        super().__init__()
        self.w1 = torch.nn.Linear(d_model, d_ff, bias=False)
        self.v = torch.nn.Linear(d_model, d_ff, bias=False)
        self.w2 = torch.nn.Linear(d_ff, d_model, bias=False)
    def forward(self, x):
        return self.w2(torch.nn.functional.silu(self.w1(x)) * self.v(x))
```
