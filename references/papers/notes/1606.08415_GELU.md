## 1. Key Innovation & Algorithmic Breakthrough

Introduces Gaussian Error Linear Unit (GELU), weighting inputs by their probability under a Gaussian distribution.

---

## 2. Core Mathematical Formulations

$$\text{GELU}(x) = x \cdot \Phi(x) = x \cdot P(X \le x), \quad X \sim \mathcal{N}(0, 1)$$

---

## 3. Architecture & Operational Mechanics

Standard activation function in BERT, GPT-2, GPT-3, ViT, and RoBERTa.

---

## 4. Code Implementation Reference

```python
import torch.nn.functional as F
activation = F.gelu(x)
```
