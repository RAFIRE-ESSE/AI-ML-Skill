## 1. Key Innovation & Algorithmic Breakthrough

Introduces Rotary Position Embedding (RoPE), encoding relative positional information by multiplying key and query vectors with a complex rotation matrix.

---

## 2. Core Mathematical Formulations

$$R_{\Theta, m}^d = \text{diag}\left(R_{\theta_1, m}, R_{\theta_2, m}, \dots, R_{\theta_{d/2}, m}\right)$$

---

## 3. Architecture & Operational Mechanics

Applies 2D rotation to 2-element sub-vectors of queries and keys depending on token index $m$.

---

## 4. Code Implementation Reference

```python
def apply_rotary_emb(xq, xk, freqs_cis):
    xq_ = torch.view_as_complex(xq.float().reshape(*xq.shape[:-1], -1, 2))
    xk_ = torch.view_as_complex(xk.float().reshape(*xk.shape[:-1], -1, 2))
    xq_out = torch.view_as_real(xq_ * freqs_cis).flatten(3)
    xk_out = torch.view_as_real(xk_ * freqs_cis).flatten(3)
    return xq_out.type_as(xq), xk_out.type_as(xk)
```
