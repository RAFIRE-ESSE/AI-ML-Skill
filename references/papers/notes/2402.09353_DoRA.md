## 1. Key Innovation & Algorithmic Breakthrough

Decomposes pretrained weights into magnitude and direction components, applying LoRA updates strictly to direction vectors.

---

## 2. Core Mathematical Formulations

$$W = m \frac{W_0 + \Delta W}{\|W_0 + \Delta W\|_c} = m \frac{W_0 + B A}{\|W_0 + B A\|_c}$$

---

## 3. Architecture & Operational Mechanics

Enables direction-only tuning closely mimicking full fine-tuning gradient behavior.

---

## 4. Code Implementation Reference

```python
# Magnitude and Direction Decomposition
magnitude = torch.norm(weight, dim=0, keepdim=True)
direction = weight / (magnitude + 1e-8)
updated_weight = magnitude * (direction + lora_b @ lora_a)
```
