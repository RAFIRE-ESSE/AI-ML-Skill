# Direct Preference Optimization (DPO) — Technical Summary

- **Title**: Direct Preference Optimization: Your Language Model is Secretly a Reward Model
- **Authors**: Rafael Rafailov, Archit Sharma, Eric Mitchell, Stefano Ermon, Christopher D. Manning, Chelsea Finn
- **Year**: 2023 (NeurIPS 2023)
- **arXiv**: [2305.18290](https://arxiv.org/abs/2305.18290)

## Key Innovation

Derives an analytical mapping between the optimal policy in RLHF and the underlying reward function, proving that the constrained reward maximization problem can be optimized **exactly** using a simple binary cross-entropy loss over preferred vs. dispreferred generation pairs $(y_w, y_l)$. Eliminates the need for training separate Reward Models or sampling from RL (PPO) loops.

---

## Loss Formulation

$$\mathcal{L}_{\text{DPO}}(\pi_\theta; \pi_{\text{ref}}) = -\mathbb{E}_{(x, y_w, y_l) \sim \mathcal{D}} \left[ \log \sigma \left( \beta \log \frac{\pi_\theta(y_w | x)}{\pi_{\text{ref}}(y_w | x)} - \beta \log \frac{\pi_\theta(y_l | x)}{\pi_{\text{ref}}(y_l | x)} \right) \right]$$

- $\pi_\theta$: Policy LLM being aligned.
- $\pi_{\text{ref}}$: Reference SFT model (frozen).
- $y_w$: Winning (preferred) output.
- $y_l$: Losing (dispreferred) output.
- $\beta$: Regularization strength parameter controlling KL drift from $\pi_{\text{ref}}$.
