## Key Innovation

Addresses the quadratic computational complexity of Transformers ($O(N^2)$) by introducing a **selective state-space model (SSM)**. By making SSM parameters input-dependent (selective scan), Mamba filters out irrelevant information dynamically while achieving **sub-quadratic $O(N)$ linear inference scaling**.

---

## Mechanism

Continuous-time state-space model parameterization:
$$h'(t) = A h(t) + B x(t)$$
$$y(t) = C h(t)$$

Discretized via zero-order hold (ZOH) with input-dependent step sizes $\Delta(x)$:
$$\bar{A} = \exp(\Delta A), \quad \bar{B} = (\Delta A)^{-1} (\exp(\Delta A) - I) \cdot \Delta B$$
$$h_t = \bar{A} h_{t-1} + \bar{B} x_t, \quad y_t = C h_t$$

- Hardware-aware parallel scan kernel computes updates directly in GPU SRAM without materializing expanded state matrices in HBM.
