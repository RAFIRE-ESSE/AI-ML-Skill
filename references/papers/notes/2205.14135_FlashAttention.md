# FlashAttention: Fast and Memory-Efficient Exact Attention with IO-Awareness — Technical Summary

- **Title**: FlashAttention: Fast and Memory-Efficient Exact Attention with IO-Awareness
- **Authors**: Tri Dao, Daniel Y. Fu, Stefano Ermon, Atri Rudra, Christopher Ré
- **Year**: 2022 (NeurIPS 2022)
- **arXiv**: [2205.14135](https://arxiv.org/abs/2205.14135)

## Key Innovation

Recognizes that standard Attention operations are **memory-bandwidth bound** (high High-Bandwidth Memory (HBM) read/write overhead). Introduces an IO-aware algorithm that uses **tiling** to compute exact softmax attention in blocks inside fast GPU SRAM without materializing the large $N \times N$ attention matrix in HBM.

---

## Core Algorithm Mechanics

1. **Tiling**: Splits inputs $Q, K, V$ into blocks that fit within GPU SRAM (typically 100-200 KB).
2. **Online Softmax**: Computes numerical Softmax incrementally per block using scaling factors:
   $$m_i = \max(m_i^{old}, \text{max}(S_i)), \quad \tilde{P}_i = \exp(S_i - m_i)$$
   Rescales running outputs on-the-fly without needing full matrix materialization.
3. **Recomputation**: In the backward pass, recalculates Softmax blocks in SRAM instead of reading $N \times N$ attention weights from HBM, reducing memory reads by 5x-20x.

---

## Performance Gains

- **Memory Overhead**: Reduced from $O(N^2)$ to $O(N)$ (linear in sequence length).
- **Speedup**: 2x-5x faster wall-clock training for standard sequence lengths (2k-8k tokens).
