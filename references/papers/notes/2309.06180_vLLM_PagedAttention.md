## 1. Key Innovation & Algorithmic Breakthrough

Introduces PagedAttention, allocating Key-Value cache memory into non-contiguous virtual pages on-demand to eliminate fragmentation.

---

## 2. Core Mathematical Formulations

$$\text{KV Cache Virtual Mapping}: \text{Logical Block Table} \to \text{Physical GPU Memory Pages}$$

---

## 3. Architecture & Operational Mechanics

Eliminates internal/external memory fragmentation, increasing LLM serving batch size and throughput 2x-4x.
