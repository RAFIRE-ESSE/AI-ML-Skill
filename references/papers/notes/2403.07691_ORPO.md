## 1. Key Innovation & Algorithmic Breakthrough

Combines Supervised Fine-Tuning (SFT) cross-entropy loss with an Odds-Ratio preference penalty directly into a single reference-model-free objective.

---

## 2. Core Mathematical Formulations

$$\mathcal{L}_{ORPO} = \mathcal{L}_{SFT} + \lambda \mathcal{L}_{OR}$$

---

## 3. Architecture & Operational Mechanics

Penalizes dispreferred generation odds relative to preferred generation odds without needing a frozen reference model in memory.

---

## 4. Code Implementation Reference

```python
odds_ratio = (prob_win / (1 - prob_win)) / (prob_lose / (1 - prob_lose))
loss = sft_loss - lambda_or * torch.log(torch.sigmoid(torch.log(odds_ratio)))
```
