## 1. Key Innovation & Algorithmic Breakthrough

Introduces Sharpness-Aware Minimization (SAM), seeking parameters in flat loss neighborhoods to improve generalization.

---

## 2. Core Mathematical Formulations

$$\min_w \max_{\|e\|_2 \le \rho} L(w + e)$$

---

## 3. Architecture & Operational Mechanics

Performs 2-step gradient updates: computing adversarial epsilon perturbation, then stepping at perturbed location.

---

## 4. Code Implementation Reference

```python
# SAM Perturbation Step
e_w = grad * rho / torch.norm(grad)
w.add_(e_w)
loss_sub = loss_fn(model(x), y)
loss_sub.backward()
w.sub_(e_w)
base_optimizer.step()
```
