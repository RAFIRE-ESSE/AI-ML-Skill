## 1. Key Innovation & Algorithmic Breakthrough

Introduces Grouped-Query Attention (GQA), partitioning query heads into groups that share single Key and Value heads.

---

## 2. Core Mathematical Formulations

$$\text{GQA Grouping}: N_q \text{ query heads} \to N_{kv} \text{ KV heads } (N_{kv} < N_q)$$

---

## 3. Architecture & Operational Mechanics

Interpolates between Multi-Head Attention ($N_{kv} = N_q$) and Multi-Query Attention ($N_{kv} = 1$) to cut KV cache VRAM by 8x.
