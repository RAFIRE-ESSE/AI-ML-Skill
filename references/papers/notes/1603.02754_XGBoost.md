- **Year**: 2016 (KDD 2016)

## Key Innovation

Presents a highly scalable end-to-end tree boosting system using exact greedy and approximate histogram split finding, regularized objective function with second-order Taylor gradients, and weighted quantile sketch algorithms for handling sparse/missing data natively.

---

## Loss Formulation

Objective function at step $t$:

$$\mathcal{L}^{(t)} \approx \sum_{i=1}^n \left[ g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i) \right] + \gamma T + \frac{1}{2} \lambda \sum_{j=1}^T w_j^2$$

- $g_i = \partial_{\hat{y}^{(t-1)}} l(y_i, \hat{y}^{(t-1)})$: First-order gradient.
- $h_i = \partial^2_{\hat{y}^{(t-1)}} l(y_i, \hat{y}^{(t-1)})$: Second-order Hessian.
- Optimal leaf weight: $w_j^* = -\frac{\sum_{i \in I_j} g_i}{\sum_{i \in I_j} h_i + \lambda}$.
